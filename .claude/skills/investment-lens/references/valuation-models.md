# 多大師估值模型詳細指南 v2.0

## ⚡ 快速模式估值選擇指南

| 公司類型 | 主要方法 | 輔助方法 | 判斷重點 |
|---------|---------|---------|---------|
| 成熟科技股 | Forward PE | EV/EBITDA | vs歷史分位數 |
| 高增長SaaS | EV/Revenue | Rule of 40 | 增長率支撐估值 |
| 週期性/硬體 | PB Ratio | ROIC vs WACC | 週期底部估值 |
| 虧損轉盈利 | 反向DCF | EV/Gross Profit | 隱含增長率合理性 |

## 1. 巴菲特視角：Owner Earnings估值法

### 核心公式
Owner Earnings = 淨利潤 + 折舊攤銷 - 維持性CapEx - 營運資本變動


### 判斷標準
| Owner Earnings倍數 | 判斷 |
|-------------------|------|
| <10x | 明顯低估 |
| 10-15x | 合理區間 |
| 15-25x | 偏貴，需高增長支撐 |
| >25x | 高估 |

## 2. 彼得·林區視角：PEG Ratio估值法

### 核心公式
PEG Ratio = 市盈率(P/E) ÷ 盈利增長率(EPS Growth %)


### 判斷標準
| PEG Ratio | 判斷 |
|-----------|------|
| <0.5 | 極具吸引力 |
| 0.5-1.0 | 有吸引力 |
| 1.0-1.5 | 合理 |
| 1.5-2.0 | 偏貴 |
| >2.0 | 昂貴 |

## 3. 反向DCF視角

### 核心邏輯
當前股價 → 反推隱含增長率 → 判斷合理性


### 判斷標準
| 隱含增長率vs合理預期 | 判斷 |
|-------------------|------|
| 低於歷史&分析師預期 | 可能被低估 |
| 接近合理預期 | 合理定價 |
| 高於預期20%以上 | 定價偏樂觀 |
| 遠超行業天花板 | 嚴重高估 |

## 4. 格林布拉特視角：魔法公式

### 核心指標
指標1：盈利收益率 = EBIT / 企業價值(EV) 指標2：投入資本回報率(ROIC) = EBIT / 投入資本


### 判斷標準
| 維度 | 優秀 | 良好 | 一般 | 差 |
|------|------|------|------|-----|
| 盈利收益率 | >8% | 5-8% | 3-5% | <3% |
| ROIC | >25% | 15-25% | 10-15% | <10% |

## 5. 達摩達蘭視角：EV/EBITDA行業對標

### 科技子行業基準
| 子行業 | 典型EV/EBITDA範圍 |
|--------|------------------|
| 大型SaaS/雲 | 20-40x |
| 成熟軟體 | 15-25x |
| 半導體 | 12-20x |
| 消費網路 | 15-30x |
| 硬體/設備 | 8-15x |

## 6. 科技股專屬視角：EV/Revenue + Rule of 40

### Rule of 40公式
Rule of 40分數 = 收入增長率(%) + 利潤率(%)


### EV/Revenue判斷基準
| EV/Revenue | 增長率>30% | 增長率15-30% | 增長率<15% |
|-----------|------------|-------------|------------|
| <5x | 極具吸引力 | 有吸引力 | 可能有原因的便宜 |
| 5-10x | 有吸引力 | 合理 | 偏貴 |
| 10-20x | 合理 | 偏貴 | 昂貴 |
| >20x | 需要極強敘事 | 泡沫風險 | 泡沫 |

### Rule of 40判斷標準
| 分數 | 判斷 |
|------|------|
| >60 | 頂級 |
| 40-60 | 健康 |
| 25-40 | 及格線以下 |
| <25 | 不及格 |

## IRR門檻檢查
| 倉位類型 | 最低IRR門檻 | 理由 |
|---------|-----------|------|
| 做多（核心） | ≥15% | 必須大幅跑贏指數 |
| 做多（戰術） | ≥20% | 短期機會成本更高 |
| 做空 | ≥20-25% | 補償無限下行風險 |

**鐵律**：「好公司+合理價格」=PASS。「好公司+有吸引力價格+IRR超門檻」=BUY。

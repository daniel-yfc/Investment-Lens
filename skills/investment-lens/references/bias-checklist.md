# 反偏見與盲區檢查清單 v2.0

## 第一部分：認知偏見自檢（6大陷阱）

### 陷阱1：確認偏誤（Confirmation Bias）
- [ ] 我是否主動搜索了看空這家公司的分析文章？
- [ ] 我的分析中，正面論點和負面論點的篇幅是否平衡？

**強制執行**：搜索 `[股票代碼] bear case` 或 `[股票代碼] short thesis`

### 陷阱2：錨定效應（Anchoring Bias）
- [ ] 我是否被歷史最高價或買入成本錨定了？
- [ ] 如果今天第一次看到這家公司，我願意以當前價格買入嗎？

**強制執行**：獨立估值，不看當前股價

### 陷阱3：敘事謬誤（Narrative Fallacy）
- [ ] 如果把「故事」完全抽掉，只看財務數字，我還會同樣看好嗎？
- [ ] 這個敘事是否有可證偽的時間節點？

### 陷阱4：從眾心理/權威偏見（Herding/Authority Bias）
- [ ] 我的看法是基於自己的分析，還是因為看到了別人的持仓推薦？
- [ ] 當前分析師評級分布如何？是否一邊倒？

### 陷阱5：處置效應（Disposition Effect）
- [ ] 如果我今天空倉，看到這家公司的全部信息，我會選擇以當前價格買入嗎？

### 陷阱6：過度自信（Overconfidence Bias）
- [ ] 我的估值假設中，是否使用了區間而非單點估計？
- [ ] 我是否給了這家公司過高的組合權重？

## 第二部分：財務紅旗檢查（7大警示信號）

### 紅旗1：收入確認異常
- [ ] 是否有會計政策變更影響收入確認時點？
- [ ] 遞延收入是在增長還是萎縮？
- [ ] 是否有大額一次性收入？

### 紅旗2：GAAP vs Non-GAAP差異過大
- [ ] Non-GAAP利潤比GAAP利潤高出多少？差距超過50%需深挖
- [ ] SBC占收入比例是否超過15-20%？
- [ ] 是否有頻繁的「一次性」調整項？

### 紅旗3：應收帳款增速 > 收入增速
- [ ] 應收帳款增長率是否大幅超過收入增長率？
- [ ] DSO是否在上升？

### 紅旗4：管理層行為異常
- [ ] 高管近期是否有大量抛售自家股票？
- [ ] 是否有關鍵高管離職？特別是CFO
- [ ] 審計師是否更換？

### 紅旗5：資本支出暴增但回報不明
- [ ] 資本支出是否大幅增加？
- [ ] 歷史CapEx是否產生了可衡量的回報？

### 紅旗6：現金流與利潤長期背離
- [ ] 是否連續多季度出現「利潤增長、現金流下降」？
- [ ] 經營活動現金流是否為負？

### 紅旗7：負債結構惡化
- [ ] 近期是否有大額債務到期？
- [ ] 信用評級是否被下調？
- [ ] 利息覆蓋倍數是否足夠？

## 第三部分：科技股特有盲區（5大盲區）

### 盲區1：TAM幻覺
TAM不等於公司真的能拿到的市場份額。合理的SAM可能只有TAM的十分之一。

### 盲區2：AI收入的真實性
- AI相關收入的具體定義是什麼？
- 是經常性的還是一次性的？
- 客戶是在試用還是已大規模部署？

### 盲區3：股票期權稀釋的隱蔽成本
- SBC占收入的百分比是多少？
- 流通股數量每年增長多少？
- 回購是否足以抵消稀釋？

### 盲區4：客戶獲取成本的拐點
- CAC趨勢是上升還是下降？
- LTV/CAC比率是否在惡化？

### 盲區5：監管尾部風險
- 是否有正在進行的反壟斷調查？
- AI監管政策的潛在影響？
- 地緣政治風險？

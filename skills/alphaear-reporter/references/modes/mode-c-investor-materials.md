# Mode C — Investor Materials

Load this file when the output is intended for an investor audience, board,
or committee: briefings, one-pagers, presentation narratives, stakeholder summaries.

---

## When to Use Mode C

- Investor briefing or one-pager for external audience.
- Board or investment committee summary.
- Presentation-ready narrative (not a full research report).
- Stakeholder-facing summary of an investment thesis.

## Tone and Style Requirements

- Executive-level: concise, confident, scannable.
- Lead with the conclusion, not the methodology.
- Avoid internal jargon and model mechanics.
- Every claim must be traceable to the underlying analysis.

## Output Structure

```
[Title] — [Company / Asset / Theme]
Valid as of: [date]

1. Executive summary (3–5 sentences max)
2. What matters now (1–3 key points)
3. Supporting evidence
4. Key risks
5. Next-step framing (what to watch, what to decide)
```

## Template

Load `assets/report-templates/investor-brief.md` as the output scaffold.
Also load `references/investor-materials-format.md` for tone and format guidelines.

## Validation

- One-pager: fits on a single page when printed at normal margins.
- No unsupported forward projections.
- Disclaimer visible if the material will be shared externally.

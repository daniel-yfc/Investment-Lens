# 6大投資哲學視角詳解 v2.0

## 🧭 智能路由系統

### 持有期導向的權重分配

```
預期持有期？
│
├─ >5年（長期核心）
│  └─ 質量複利(50%) + 想像力成長(30%) + 其他(20%)
│
├─ 1-3年（中期戰術）
│  └─ 基本面多空(40%) + 催化劑驅動(30%) + 質量複利(20%) + 其他(10%)
│
├─ <1年（短期交易）
│  └─ 催化劑驅動(50%) + 宏觀戰術(30%) + 其他(20%)
```

## 1. 質量複利視角（Quality Compounder）

**哲學**：「以合理價格買入絕妙的生意，遠勝以絕妙價格買入平庸的生意。」

**適用對象**：
- **個股**：高ROIC、寬護城河的優質公司
- **市場/ETF**：成熟市場指數（VOO、VTI），關注長期盈利增長

**關鍵指標**：ROIC長期趨勢（個股）、ROE與盈利增長（市場指數）

**執行紀律**：
- **個股**：連續3年ROIC>15%，護城河可持續，管理層理性
- **市場**：長期盈利增長穩定，估值合理，制度環境良好

**核心問題**：如果市場關閉10年，你能安心持有嗎？

---

## 2. 想像力成長視角（Imaginative Growth）

**哲學**：「最大的投資錯誤是想像力不足。」

**適用對象**：
- **個股**：處於S曲線加速期的高增長公司
- **產業/區域**：結構性成長趨勢（AI、印度人口紅利、能源轉型）

**關鍵指標**：TAM可實現性、滲透率曲線位置

**執行紀律**：
- 收入增速>30%，Unit Economics健康
- TAM足夠大且可驗證
- 管理層有執行大願景的能力

**核心問題**：5年後回頭看，今天不買會後悔嗎？

---

## 3. 基本面多空視角（Fundamental Long/Short）

**哲學**：「用深度研究找到市場定價錯誤的標的。」

**適用對象**：
- **個股**：被錯誤定價的公司（高估做空，低估做多）
- **產業輪動**：週期性行業、主題輪換

**關鍵指標**：Variant Perception（與市場共識的差異）

**執行紀律**：
- 必須有清晰的「為什麼市場錯了」
- 明確的催化劑和時間框架
- 風險回報不對稱（下有底、上有天花板）

**核心問題**：有沒有市場還沒看到的Variant View？

---

## 4. 深度價值視角（Deep Value）

**哲學**：「買別人在賣的、不受待見的、被嫌棄的東西。」

**適用對象**：
- **個股**：困境反轉、資產重估
- **市場/區域**：被忽視的市場、週期底部

**關鍵指標**：安全邊際、重置成本、清算價值

**執行紀律**：
- 最低30%安全邊際，理想50%+
- 必須有催化劑，避免價值陷阱
- 高度分散（因為個別標的可能歸零）

**核心問題**：當前價格比清算/重置成本低多少？

---

## 5. 催化劑驅動視角（Event-Driven）

**哲學**：「識別會觸發市場重新定價的具體事件。」

**適用對象**：
- **個股**：併購、分拆、重組、新產品發布
- **市場**：政策變化、央行決議、地緣事件

**關鍵指標**：事件概率、影響幅度、時間確定性

**執行紀律**：
- 事件必須具體、可驗證、有影響力
- 概率加權分析
- 明確的退出條件

**核心問題**：未來6-18個月，有什麼具體事件會改變定價？

---

## 6. 宏觀戰術視角（Macro-Tactical）

**哲學**：「推動市場的不是盈利，是流動性和週期。」

**適用對象**：
- **所有資產類別**：根據宏觀環境調整配置權重
- **大類資產配置**：股債商品的戰術性調整

**關鍵指標**：
- 參考 `macro-liquidity.md`：Fed淨流動性、利率、通膨
- 經濟週期位置、政策取向

**執行紀律**：
- 識別當前宏觀環境（復甦/過熱/滯脹/衰退）
- 配置有利於當前環境的資產
- 設定環境變化的觸發條件

**核心問題**：當前宏觀環境對這個資產是順風還是逆風？

## 視角衝突處理原則

當多個視角給出不同結論時：

| 衝突類型 | 解讀 | 處理方式 |
|---------|------|---------|
| 質量複利(Buy) vs 宏觀戰術(Sell) | 好公司遇壞時機 | 降低倉位或等待更好時機 |
| 想像力成長(Buy) vs 深度價值(Pass) | 增長vs估值紀律衝突 | 取決於風險偏好和持有期 |
| 基本面多空(Buy) vs 催化劑(Pass) | 有觀點但缺觸發點 | 可建倉但預期持有期更長 |

**解決原則**：
1. 長期持有（>3年） → 質量複利和想像力成長優先
2. 中期持有（1-3年） → 基本面多空和催化劑驅動優先  
3. 短期交易（<1年） → 催化劑驅動和宏觀戰術優先

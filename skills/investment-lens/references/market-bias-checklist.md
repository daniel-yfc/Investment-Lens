# 市場與資產配置反偏見檢查清單 v1.0

本文件是資產配置決策的「安全網」。在完成市場分析和配置建議後，必須逐條檢查這個清單。目的不是讓你不投資，而是確保決策是基於理性分析，而非情緒和偏見。

---

## 第一部分：宏觀認知偏見自檢（7大陷阱）

### 陷阱1：本土偏好（Home Country Bias）

**什麼意思**：過度投資於自己熟悉的市場。台灣人80%資產在台股，美國人只買美股，忽略全球分散化的好處。

**自檢問題**：
- [ ] 我在單一國家/區域的配置是否超過其全球市值佔比的2倍？
- [ ] 如果本國貨幣大幅貶值20%，我的總資產購買力會損失多少？
- [ ] 我重倉這個市場是因為客觀分析，還是因為「感覺熟悉」？
- [ ] 我是否認真評估過其他區域的投資機會？

**強制執行**：
- 搜索：`global stock market capitalization by country`
- 計算：自己的配置 vs 全球市值加權基準（VT）的偏離度
- 若單一國家配置 > 其全球市值佔比的2倍，需要強力理由

**全球市值參考**：美國~60%、歐洲~15%、日本~6%、中國~3%、台灣~2%

### 陷阱2：線性外推謬誤（Linear Extrapolation）

**什麼意思**：假設過去的趨勢會永遠持續。看到美股過去10年年化12%，就假設未來也如此。

**自檢問題**：
- [ ] 支撐過去表現的宏觀條件（低利率、QE、全球化）現在還存在嗎？
- [ ] 如果未來10年回報只有過去的一半，我還願意持有嗎？
- [ ] 我是否只看了「績效圖很漂亮」的時間段？
- [ ] 機構對該資產未來10年的預期回報是多少？

**強制執行**：
- 搜索：`[資產] long term return expectations institutional`
- 對比：過去10年實際回報 vs 機構預期未來10年回報
- 如果差異巨大，需要解釋原因

### 陷阱3：單一敘事謬誤（Single Narrative Fallacy）

**什麼意思**：用一個簡單故事解釋複雜世界。「AI是未來所以買QQQ」、「通膨來了所以買黃金」。

**自檢問題**：
- [ ] 我的投資理由是否可以濃縮成一句話？（如果是，可能過度簡化）
- [ ] 是否存在明顯反例？（如高通膨時黃金也曾下跌）
- [ ] 我是否考慮了估值、匯率、政策等複合因素？
- [ ] 這個敘事是否已經被市場充分定價？

**強制執行**：
- 列出至少2個可能破壞主要敘事的「破壞性因子」
- 分析：「敘事層面論點」vs「數據支撐的證據」

### 陷阱4：收益率追逐（Yield Chasing）

**什麼意思**：在低利率環境下盲目追求高配息，忽略總回報和本金風險。

**自檢問題**：
- [ ] 這個高配息是否來自本金返還或衰退產業？
- [ ] 扣除通膨、稅收、匯率損失後，真實收益率是多少？
- [ ] 我關注的是現金流入感覺，還是長期總回報？
- [ ] 配息的可持續性如何？

**強制執行**：
- 計算：真實收益率 = 名目配息率 - 通膨率 - 稅率 - 匯率風險
- 檢查：過去5年的總回報（含息）vs 僅配息回報

### 陷阱5：錨定效應（宏觀版）

**什麼意思**：被歷史高點/低點錨定判斷。「黃金從2000跌到1700，應該便宜了」。

**自檢問題**：
- [ ] 如果不知道歷史價格，只看估值指標，我還會買嗎？
- [ ] 我的判斷基於價格位置，還是價值評估？
- [ ] 歷史高點/低點時的宏觀環境和現在可比嗎？
- [ ] 我是否在用「跌了X%」說服自己，而非用「估值處於Y分位」？

**強制執行**：
- 盲測：隱藏價格圖，僅看估值指標（PE、PB、ERP等）
- 判斷：先得出結論，再看價格，檢查是否被錨定影響

### 陷阱6：近因偏誤（Recency Bias）

**什麼意思**：過度重視最近表現，忽略長期週期。「科技股過去3年漲100%，一定要配置」。

**自檢問題**：
- [ ] 我是否因最近3年的表現而改變長期配置？
- [ ] 這個資產是否處於週期極端位置？
- [ ] 我是否在「追漲殺跌」？
- [ ] 完整週期（10年+）的表現如何？

**強制執行**：
- 查看：滾動10年回報，識別週期性模式
- 對比：最近3年 vs 過去10年 vs 滾動10年最差/最佳表現

### 陷阱7：過度自信與控制幻覺

**什麼意思**：高估自己預測市場轉折的能力。「我會在高點賣出」。

**自檢問題**：
- [ ] 我過去的擇時記錄如何？（誠實記錄）
- [ ] 我是否高估了預測宏觀轉折的能力？
- [ ] 如果我錯了，代價是什麼？
- [ ] 我應該用資產配置再平衡取代主動擇時嗎？

**強制執行**：
- 記錄：過去12個月的重大配置決策及結果
- 統計：正確率和平均影響
- 如果正確率<60%，應該轉向被動配置

## 第二部分：ETF/基金特有風險掃描（5大紅旗）

### 紅旗1：隱藏的集中度風險

**檢查**：前10大持股佔比、單一股票權重上限

**警戒標準**：
- 單一持股 >15% → 🔴 高風險（如0050的台積電）
- 前10大持股 >50% → ⚠️ 需關注
- 前5大持股 >40% → 🔴 嚴重集中

**實例警示**：
- 0050：台積電權重45%+，實質是「台積電+供應鏈ETF」
- QQQ：科技七巨頭合計>30%，並非真正的科技分散
- VOO：科技股集中度創歷史新高

### 紅旗2：匯率風險被忽視

**檢查**：是否有匯率對沖、本幣vs標的貨幣走勢

**風險評估**：
- 無對沖海外ETF：承擔完整匯率波動
- 新興市場基金：貨幣貶值風險高
- 美元計價ETF：美元強弱直接影響回報

**實例**：台灣人買VOO，實際回報 = 美股回報 + 美元兌台幣匯率變動

### 紅旗3：費用與追蹤誤差

**檢查**：
- [ ] ETF規模是否 <1億美元？（清算風險）
- [ ] 日均成交量是否過低？
- [ ] 溢價/折價是否經常 >1%？
- [ ] 槓桿ETF是否存在「波動率耗損」？

### 紅旗4：商品ETF的期貨結構風險

**適用**：原油（USO）、天然氣（UNG）等期貨連結ETF

**問題**：長期Contango（正價差）結構會侵蝕回報
- 即使現貨價格不變，ETF也可能因換倉成本而下跌
- 需檢查：期貨曲線形狀、歷史滾動成本

### 紅旗5：高股息ETF的陷阱

**檢查**：
- [ ] 高配息來源是否可持續？
- [ ] 是否集中在衰退產業？（煙草、公用事業）
- [ ] 是否使用複雜策略？（如賣出買權）
- [ ] 總回報是否跑輸大盤？

## 第三部分：Pre-Mortem事前屍檢（宏觀版）

想像2年後，你的主要配置虧損30%以上。最可能的原因：

### 問題1：最可能的失敗路徑

**範例：VOO（美股大盤）失敗分析**

#### 路徑A：AI泡沫破裂
- **場景**：AI商業化不及預期，科技巨頭估值回歸正常
- **驅動**：監管加強、技術瓶頸、競爭加劇
- **早期信號**：科技股CapEx/Revenue上升但收入增速放緩
- **Kill Condition**：科技七巨頭PE從30倍跌破20倍 → 減倉50%

#### 路徑B：利率長期高位
- **場景**：通膨頑固，Fed維持5%+利率3年
- **驅動**：財政赤字、供應鏈重組推高結構性通膨
- **早期信號**：ERP<2%持續6個月、10年期收益率>5%
- **Kill Condition**：ERP<1.5%連續3個月 → 減倉30%

#### 路徑C：地緣政治危機
- **場景**：重大軍事衝突、全球化逆轉
- **驅動**：台海危機、美中脫鉤加速
- **早期信號**：VIX持續>30、美元指數急升
- **Kill Condition**：全球貿易量YoY下降>10% → 轉向防禦配置

### 問題2：核心假設脆弱性

| 我的核心假設 | 如果錯了的後果 | 被證偽可能性 |
|------------|---------------|-------------|
| 美股長期年化8-10% | 配置過高，實際回報遠低於預期 | 中（歷史均值回歸） |
| AI將推動生產力革命 | 科技股泡沫，組合重創 | 中（技術發展不確定） |
| 美元霸權持續 | 匯率損失，國際配置失效 | 低（短期內難以撼動） |

### 問題3：聰明的反對者可能看到什麼？

**強制執行**：
- 搜索：`[標的] bear case strongest argument`
- 搜索：`[標的] bubble concerns valuation risks`
- 找出最尖銳的反對意見，並在報告中回應

## 第四部分：使用指南

完成檢查後，將結果分類：

### 🔴 紅旗警報
- 發現明確且嚴重的風險信號
- 必須在配置中予以重大考量
- 需要設置嚴格的Kill Condition

### 🟡 黃旗提醒  
- 存在潜在風險，需要監控
- 可能影響持有時長或倉位上限
- 設定明確的監控指標

### ✅ 綠燈確認
- 經主動驗證後的積極因素
- 不是「沒發現問題」，而是「刻意檢查後仍然安全」
- 支持當前配置決策

**每項檢查都要附帶證據，而非簡單的通過/不通過。**

## 🎯 最終靈魂拷問

在執行配置調整前，問自己：

> **「如果未來3年這個資產完全沒有上漲，甚至下跌10%，我依靠什麼獲利？是再平衡？是股息再投入？還是我會恐慌賣出？」**

> **「如果我今天空倉，看到所有信息，我會選擇以當前價格買入嗎？」**

> **「這個決策是基於價值判斷，還是被價格位置、近期表現或情緒影響？」**

如果答案讓你不安，說明配置可能過於激進，需要調整至你能長期堅持的水平。

**記住**：最好的資產配置不是回報最高的，而是你能長期堅持的。控制能控制的（成本、情緒、紀律），接受不能控制的（市場波動）。

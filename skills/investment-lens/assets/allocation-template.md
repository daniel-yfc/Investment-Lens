# 📋 個人資產配置輸入格式引導

## 🚦 格式選擇指引

| 你的情況 | 建議格式 |
|---------|---------|
| 退休規劃、有明確提領時間點 | **格式 A：退休導向版** |
| 目標導向配置（買房、教育、創業） | **格式 B：目標導向版** |
| 快速了解現況與建議方向 | **格式 C：快速版** |
| 剛開始投資、資料不完整 | **格式 D：對話式** |

> 💡 **注意**：本模板供個人配置規劃使用，輸出為分析框架，非個人化受規範之投資建議。

---

## 格式 A：退休導向版

```
# ===================================================
# value_date:    [YYYY-MM-DD]
# allocation_mode: retirement
# ===================================================

【個人基本資料】
- 年齡：[X] 歲
- 預計退休年齡：[X] 歲（距今 [X] 年）
- 預計壽命規劃至：[X] 歲
- 年收入稅率級距：[X]%
- 主要幣別：TWD / USD / other
- 是否已有勞保 / 勞退 / 國民年金：是 / 否（月領估算 NT$[X]）
- 是否有其他穩定收入（租金、版稅等）：是 / 否（月收約 NT$[X]）

【財務狀況】
- 目前可投資總資產：NT$[X]
- 緊急預備金（現金）：NT$[X]（約 [X] 個月生活費）
- 每月可新增投入：NT$[X]
- 預計退休後每月提領：NT$[X]
- 提領方式偏好：定額提領 / 定比例提領 / 彈性提領

【風險承受度】
- 自評風險偏好：保守 / 穩健 / 積極
- 可接受的最大年度帳面虧損：[X]%
- 上次市場大跌（如 2022 年）時的反應：賣出 / 持有 / 加碼

【現有配置概況（可選填）】
- 股票類：約 [X]%（NT$[X]）
- 債券類：約 [X]%（NT$[X]）
- 現金 / 貨幣市場：約 [X]%（NT$[X]）
- 另類資產（黃金、REITs、商品）：約 [X]%（NT$[X]）
- 其他：約 [X]%（NT$[X]）

【稅務考量（可選填）】
- 是否在意二代健保補充保費：是 / 否
- 海外所得年度估算：NT$[X]（是否接近 750 萬門檻）
- 遺產規劃需求：有 / 無
```

## 格式 D：對話式

```
我 [X] 歲，[職業/狀況簡述]。
目前有約 NT$[X] 可以投資，緊急預備金另外存著。
主要目標是 [退休 / 購屋 / 其他]，希望在 [X] 年內達成。
風險偏好偏 [保守 / 穩健 / 積極]，對虧損的接受度大概是 [X]%。

目前的配置大概是：[簡單描述，例如：大部分放台股 ETF，少部分現金]

想了解：[你的問題，例如：我現在的配置合理嗎？退休後要怎麼提領？]
```

## 🔍 欄位說明

### 風險承受度評估參考

| 類型 | 可接受最大年度虧損 | 典型股債比 | 適合情境 |
|------|-----------------|-----------|---------|
| 保守 | ≤ 10% | 30/70 | 退休後、短期目標（< 3 年） |
| 穩健 | 10–20% | 50/50 至 60/40 | 中期目標（3–10 年）、退休前 5 年 |
| 積極 | 20–35% | 70/30 至 90/10 | 長期目標（> 10 年）、年輕、收入穩定 |

### 提領率參考

| 提領率 | 說明 | 風險 |
|--------|------|----- |
| 3% | 保守提領，資產長期可持續 | 低 |
| 4% | 4% 法則，歷史上多數情況下可持續 30 年 | 中 |
| 5% | 較積極，需搭配彈性提領策略 | 較高 |
| > 5% | 高提領，長壽風險上升 | 高 |

> 台灣投資人補充：勞退月領可降低提領率需求。確認勞退帳戶估算金額後，可用「可投資資產 × 提領率 + 月領勞退」計算退休月收入。

### 三桶法資金分配

| 桶 | 功能 | 建議資產 | 規模建議 |
|----|------|---------|----------|
| 安全桶 | 緊急備用 + 近期提領 | 活存、定存、貨幣市場 | 1–3 年生活費 |
| 收入桶 | 中期穩定現金流 | 投資等級債、高股息 ETF | 3–7 年預計提領 |
| 成長桶 | 長期資本增值 | 全球股票 ETF、另類資產 | 剩餘資產 |

## ⚠️ 注意事項

- 本模板產出的配置建議為分析框架，非受規範之個人投資建議。
- 台灣投資人應將勞保年金、勞退新制帳戶視為「隱形債券」計入整體配置。
- 配置建議應每 1–3 年或重大生活事件發生時（退休、失業、重大醫療支出）重新檢視。
- 若有稅務或法律問題，建議諮詢合格的稅務顧問或理財規劃師。

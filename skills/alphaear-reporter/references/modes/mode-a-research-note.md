# Mode A — Research Note

Load this file when the output is a short or medium-form research document:
internal memo, thesis update, signal summary, or quick analytical note.

---

## When to Use Mode A

- Thesis update after earnings or a macro event.
- Signal summary from AlphaEar signal pipeline.
- Internal memo for a single investment decision.
- Short-form note that does not require the full initiating coverage format.

## Required Source Inputs

- Analysis output from `investment-lens` (Mode A, B, or D) or signal skills.
- A clearly stated recommendation or view.
- Valid-as-of date.

## Output Structure

```
Title
Valid as of: [date] | Confidence: [High / Medium / Low]

1. What changed (or: situation summary)
2. Core view
3. Key evidence (2–4 points)
4. Risks and counter-arguments
5. Monitoring conditions
```

## Template

Load `assets/report-templates/research-note.md` as the output scaffold.

## Length

- Target: 300–800 words.
- If the analysis source is richer, a longer note is acceptable but must
  preserve the above structure.

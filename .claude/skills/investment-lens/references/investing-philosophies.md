# 6大投資哲學視角詳解 v3.0

## 🧭 投資哲學智能路由系統

### 主導視角選擇決策樹
持有期預期？ │ ├─ >5年（長期核心） │ └─ 質量複利(50%) + 想像力成長(30%) + 其他(20%) │ ├─ 1-3年（中期戰術） │ └─ 基本面多空(40%) + 催化劑驅動(30%) + 質量複利(20%) + 其他(10%) │ ├─ <1年（短期交易） │ └─ 催化劑驅動(50%) + 宏觀戰術(30%) + 其他(20%) │ └─ 未明確 → 默認中期權重


### 特殊情境權重調整
| 情境 | 權重調整 | 觸發條件 |
|------|---------|---------|
| **宏觀極端環境** | 宏觀戰術 → 50%+ | VIX>30、流動性危機 |
| **明顯錯殺股** | 深度價值 → 40%、質量複利 → 30% | 基本面良好但股價暴跌>30% |
| **高增長未盈利** | 想像力成長 → 50%、催化劑 → 30% | 收入增速>50%、GAAP虧損 |

## 1. 質量複利視角（Quality Compounder）
**哲學**：「以合理價格買入絕妙的生意，遠勝以絕妙價格買入平庸的生意。」
**代表人物**：巴菲特、芒格、李錄
**時間範圍**：永久持有

### 執行紀律
- [ ] 連續3年ROIC是否>15%？
- [ ] 如ROIC<10%且未改善 → 自動排除長期核心候選
- [ ] 護城河是否有量化證據？

**核心問題**：如果市場關閉10年，你拿著這隻股票能安心睡覺嗎？

## 2. 想像力成長視角（Imaginative Growth）
**哲學**：「最大的投資錯誤是想像力不足。」
**代表人物**：Baillie Gifford、ARK
**時間範圍**：5年以上

### 執行紀律
- [ ] TAM的可實現性如何驗證？
- [ ] 增長的可持續性依賴什麼？
- [ ] 管理層有執行大願景的能力嗎？

**核心問題**：5年後回頭看，今天不買會不會後悔？

## 3. 基本面多空視角（Fundamental Long/Short）
**哲學**：「用深度研究買入最好的公司，做空最差的公司。」
**代表人物**：Tiger Cubs
**時間範圍**：1-3年

### 執行紀律
**Variant Perception必須歸類為**：
1. **數字認知差異**：市場低估/高估某財務指標含義
2. **估值框架差異**：市場用錯估值方法
3. **時間節奏差異**：市場對變化時間預期錯誤

**核心問題**：有沒有一個市場還沒看到的Variant View？

## 4. 深度價值視角（Deep Value）
**哲學**：「我專門買別人在賣的、不受待見的、被嫌棄的東西。」
**代表人物**：Seth Klarman、Howard Marks
**時間範圍**：數年

### 執行紀律
- 估值錨點：淨現金+有形資產 > 重置成本 > 正常化盈利能力
- 安全邊際要求：最低30%折扣，理想50%+

**核心問題**：市場出價比這家公司的「清算價值」低多少？

## 5. 催化劑驅動視角（Event-Driven）
**哲學**：識別會觸發市場重新定價的具體事件
**代表人物**：David Tepper、Bill Ackman
**時間範圍**：6-18個月

### 執行紀律
**催化劑必須滿足**：
1. **具體性**：明確事件名稱和時間窗口
2. **可驗證性**：結果可客觀判斷
3. **影響力**：對股價有可量化預期影響

**核心問題**：未來6-18個月，有什麼具體事件會迫使市場重新定價？

## 6. 宏觀戰術視角（Macro-Tactical）
**哲學**：「推動整體市場的不是盈利，是美聯儲。」
**代表人物**：Stanley Druckenmiller
**時間範圍**：隨宏觀週期

### 執行紀律
**宏觀環境評估框架**：
| 維度 | 權重 | 對科技股影響 |
|------|------|-------------|
| 流動性 | 40% | 擴張有利/收縮不利 |
| 利率 | 30% | 降息有利/升息不利 |
| 經濟週期 | 20% | 擴張有利/衰退不利 |
| 風險偏好 | 10% | 高偏好有利/低偏好不利 |

**核心問題**：當前宏觀環境對這隻股票是順風還是逆風？

## 視角衝突處理矩陣
| 衝突類型 | 解讀 | 建議 |
|---------|------|------|
| 質量複利(Buy) vs 宏觀戰術(Sell) | 好公司遇壞時機 | 降低倉位或等待 |
| 想像力成長(Buy) vs 深度價值(Pass) | 增長vs估值紀律 | 取決於風險偏好 |
| 基本面多空(Buy) vs 催化劑(Pass) | 有觀點缺觸發點 | 可建倉但預期更長 |
